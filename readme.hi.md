इसलिए मैंने उन्हें एक अलग फ़ोल्डर में स्थानांतरित करने का विचार किया, ताकि मैं अपने सभी उपकरणों पर एक ही प्लगइन्स का उपयोग कर सकूँ।

फिर सोचा: 'अगर मैं यह कर ही रहा हूँ, तो क्यों न इसे GitHub पर डाल दूँ, ताकि कहीं से भी एक्सेस किया जा सके — और शायद यह दूसरों के लिए भी उपयोगी हो सकता है।'

इसके बाद क्लासिक प्रोग्रामर सिंड्रोम शुरू हुआ: *'यह Renovate के साथ प्रबंधनीय होना चाहिए — अगर मैं प्लगइन्स को मशीन-पठनीय बनाऊँ और उन्हें स्वचालित रूप से डाउनलोड करूँ।'*
