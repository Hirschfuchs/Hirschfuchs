### 🔏 KeePass 外掛程式集合 ([查看倉庫](https://github.com/Hirschfuchs/keepass-plugins))

> 個人收集的實用 KeePass 外掛程式，與大家分享，並提供自動更新。

#### NEU
在使用 KeePass 資料庫時，我注意到 KeePass 主程式可以透過套件管理工具更新，但外掛程式需要手動維護。於是我想到將外掛程式移到一個獨立的資料夾，這樣我就可以在不同設備間使用相同的外掛程式。

然後我又想：既然如此，不如將它放到 GitHub 上，這樣我可以隨時訪問，也可以與其他人分享，或許有人會覺得有用。

接著，我陷入了典型的程式設計師症候群：*「如果將外掛程式做成機器可讀並自動下載，那就可以用 Renovate 來維護了。」*

於是，這個專案就誕生了。目前進度大約 80%，我對結果很滿意。:)

- 🚆 Deutschlandticket 規劃器: 計算使用 Deutschlandticket 在設定的最大旅行時間內可行的火車路線
